---
name: run-playwright
description: Force a Playwright MCP browser screenshot of the current dev server
---

Use the Playwright MCP to take a browser screenshot:

1. Identify the dev server URL — check package.json scripts, or ask if unsure (default: http://localhost:3000)
2. Take a screenshot via the Playwright MCP tool
3. Scale to 75%, PNG format
4. Describe what you see: layout, colours, spacing, any visual issues
5. If a problem is visible, propose a fix
