## Standards & Conventions
 - An advanced and modern approach to mobile first responsiveness. 
 - Adhearing to all web standards and w3c core requirements for english site. 
 - Keep presentation, functionality and content as separate
 - Focus on performance, fast loading, 100% pagespeed score, fast image loading retina images served to retina devices. 
 - light/dark mode styles. 
 - Fonts in em and dimentions in rem. 
 - Use clean css for modern browsers and use variables for things like colours. 
 - Comprehensive global styles needed for optimum ux: well organised standard practices for typography, forms, buttons, hover / active states, tooltips, modal windows, alerts, svg icons. 
 - Accessibility needs and keyboard access should be catered for. 
 - Ensure all features needed for mobile device usage such as orientation settings. 
 - UX should be straight-forward, clean and minimal - keep text, buttons and form fields large and color scheme simple. 
 - SEO should be optimised, Schemas included
